# GECAT Website

Site estático bilíngue do GECAT/UFES.

## Estrutura correta

- `index.html` — página inicial em português
- `research.html` — pesquisa
- `people.html` — pessoas
- `publications.html` — publicações
- `working-papers.html` — working papers
- `awards.html` — prêmios
- `applied-projects.html` — projetos aplicados
- `contact.html` — contato
- `en/` — versão em inglês
- `assets/` — CSS, JavaScript e imagens
- `data/` — dados auxiliares

Para publicar no GitHub Pages, envie o conteúdo deste ZIP diretamente na raiz do repositório.
