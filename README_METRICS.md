# Journal metrics and impact indicators

The publications page uses available journal indicators in the following order:

1. SJR / SCImago quartile when available
2. JCR / Journal Citation Reports indicator when available
3. ABS/AJG when available
4. Qualis/CAPES historical information
5. CAPES / Periódicos-ADM 2025–2028 as a complementary source when the other indicators are missing or incomplete

The color code is:

- Green: higher relative impact based on the strongest available indicator
- Yellow: intermediate or moderate impact
- Red: low impact (not currently used in the dataset)
- Gray: classification needs confirmation

Main complementary source:
https://periodicos-adm.com/
